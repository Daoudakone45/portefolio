# Portfolio Professionnel - Daouda Kone

## 🚀 À Propos

Portfolio design moderne et professionnel mettant en avant mes compétences en **développement web Full Stack**. Construit avec **Tailwind CSS** pour un design corporatif élégant et responsive.

## 🎯 Contenu du Portfolio

### 📌 Sections Principales

1. **Accueil (Hero)** - Présentation impactante avec CTA
2. **À Propos** - Parcours, objectifs professionnels et académiques
3. **Compétences** - Catégories Frontend/Backend/DevOps avec barres de progression
4. **Projets** - 3 projets phares avec technologies utilisées
5. **Expérience** - Timeline chronologique des expériences pro et académiques
6. **Contact** - Formulaire + moyens de contact + réseaux sociaux
7. **Footer** - Crédits et informations légales

---

## 🛠️ Stack Technologique

### Frontend
- **HTML5** - Structure sémantique
- **Tailwind CSS** - Framework CSS utility-first
- **JavaScript Vanilla** - Interactivité (menu mobile, animations, formulaire)

### Fonctionnalités
- ✅ Menu mobile responsive
- ✅ Smooth scrolling navigateur
- ✅ Animations au scroll (fade-in)
- ✅ Formulaire de contact avec validation
- ✅ Barres de compétences animées
- ✅ Icônes Font Awesome 6.4
- ✅ Design adaptatif (mobile-first)

## 🚀 Démarrage Rapide

### Sans serveur (fichier local)
```bash
# Ouvrir directement index.html dans le navigateur
open index.html
```

### Avec Node.js/Express (optionnel)
```bash
# Si vous avez créé un server.js
node server.js
# Le portfolio sera accessible sur http://localhost:3000
```

## ✨ Fonctionnalités Interactives

### 1. Menu Mobile
- Toggle au clic sur l'hamburger icon
- Fermeture automatique au clic sur un lien
- Fermeture au clic extérieur

### 2. Formulaire de Contact
- Validation des champs (nom, email, sujet, message)
- Message de succès avec transition
- Reset automatique après 3 secondes

### 3. Animations
- Fade-in des sections au scroll
- Barres de compétences animées (0% → valeur finale)
- Effets hover sur les cartes (translateY)
- Smooth transitions sur tous les éléments

### 4. Navigation
- Smooth scrolling avec offset pour la nav fixe
- Liens de section actifs
- Scrollbar personnalisée
